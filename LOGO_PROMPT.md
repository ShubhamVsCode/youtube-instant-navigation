## Key Design Elements to Emphasize

1. **Circular shape** - Standard Chrome extension icon format
2. **YouTube red** - Brand recognition (#FF0000)
3. **White play button** - YouTube iconography
4. **Golden keyboard keys** - Represents shortcuts (ESC, 1, J)
5. **Minimalist** - Must work at 16px size
6. **High contrast** - Readable at small sizes
7. **Flat design** - Modern, clean aesthetic

## Color Palette Reference

- **Background**: #FF0000 (YouTube Red) to #CC0000 (Darker Red)
- **Play Button**: #FFFFFF (White)
- **Keyboard Keys**: #FFD700 (Gold) or #D4A853 (Golden)
- **Key Text**: #1A1A1A (Dark Gray/Black)

## Tips for Best Results

1. **Iterate**: Generate multiple variations and pick the best
2. **Test at small sizes**: Make sure it's readable at 16x16
3. **Keep it simple**: Too many elements won't work at small sizes
4. **High contrast**: Ensure keys are visible against red background
5. **Consistent style**: All keys should have the same visual weight

## Post-Processing

After generating:
1. Export at exact sizes: 16x16, 48x48, 128x128
2. Test visibility at 16px
3. Adjust contrast if needed
4. Ensure circular shape is perfect
5. Remove any text that might be too small
